# 写作风格指南

> 本文件由用户和 Agent 共同维护。Agent 在撰写任何章节前必须阅读此指南。
> 用户可随时补充新规则，Agent 在审校时也可提议新增条目。

---

## 1. 段落结构

- 每段以**主题句**（Topic Sentence）开头，概括本段核心观点
- 先结论后论据：先陈述发现/判断，再给出支撑细节
- 单段长度控制在 5-10 句以内，避免过长段落

## 2. 数字与百分比

- 10 以下的数字用汉字（"三个模型"），10 及以上用阿拉伯数字（"12 个样本"）
- 百分比统一使用 "X%" 格式（如 "准确率提升了 3.2%"）
- 小数点前的零不可省略（"0.85" 而非 ".85"）

## 3. 图表引用措辞

- 引用图表时使用"如图 X 所示"、"表 X 列出了……"
- 避免"从图 X 可以看出"等主观表述，改用"图 X 表明……"
- 所有图表必须在正文中至少引用一次

## 4. 禁止使用的套话

以下短语因过于空泛而禁止使用：
- "近年来，……引起了广泛关注"
- "众所周知"
- "不言而喻"
- "毫无疑问"
- "在实际应用中具有重要意义"

## 5. 中文学术写作规范

- 使用中文标点符号（全角逗号、句号、括号等）
- 中英文之间加一个空格（"使用 Transformer 架构"）
- 数字与中文之间加一个空格（"共 12 个样本"）
- 术语首次出现时附注英文原文，格式：`中文术语（English Term, 缩写）`

## 6. 过渡与衔接

- 章节之间使用过渡段落，简要回顾上文并引出下文
- 段落之间使用适当的连接词（"此外"、"然而"、"具体而言"等）
- 避免突兀的话题切换

## 7. 各章节长度指导

（由初始化时根据投稿目标和页数限制自动生成，以下为默认建议）

- 摘要：150-300 字
- 引言：1-2 页
- 相关工作：1-1.5 页
- 方法：2-3 页
- 实验：1-2 页
- 结论：0.5-1 页
