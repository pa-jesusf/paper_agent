# 论文大纲

> 此文件是论文撰写的核心路线图。Agent 在撰写任何章节前必须参考此大纲。
> 章节结构不预设固定模板，由用户和 Agent 在初始化时共同确定。
> `paper/sections/` 中的 `.tex` 文件和 `paper/main.tex` 中的 `\input{}` 顺序
> 必须与此大纲保持同步。
