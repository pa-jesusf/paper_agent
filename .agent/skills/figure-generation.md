# Skill: 图表生成

> 从数据或描述生成论文级图表。

## 触发条件

用户请求生成图表、可视化数据、绘制架构图等。

## 工作步骤

1. 读取 `config/figure-style.yaml` 获取视觉风格配置
2. 如果是首次生成图表且用户有已有图表，先分析已有风格
3. 生成图表，保存到 `pipeline/figures/fig_{章节}_{描述}.{格式}`
4. 生成 `_meta.yaml` 元信息文件
5. 如需在论文中引用，复制到 `paper/figures/` 并更新对应 `.tex` 文件

## 风格要求

- 配色方案、字体、DPI 等必须从 `figure-style.yaml` 读取
- 优先继承用户已有图表的视觉风格
- 所有图表保持统一的视觉语言
