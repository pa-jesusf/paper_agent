# Skill: 图表生成

> 从数据或描述生成论文级图表。

## 触发条件

用户请求生成图表、可视化数据、绘制架构图等。

## 工作步骤

1. 尝试读取 `config/figure-style.yaml` 获取视觉风格配置（不存在则跳过）
2. 如果是首次生成图表且用户有已有图表，可先分析已有风格
3. 生成图表，保存到 `pipeline/figures/fig_{章节}_{描述}.{格式}`
4. 生成 `_meta.yaml` 元信息文件
5. 如需在论文中引用，复制到 `paper/figures/` 并更新对应 `.tex` 文件

## 风格要求

- 配色方案、字体、DPI 等优先从 `figure-style.yaml` 读取。**配置文件不存在时**，使用学术期刊通用默认值（如 300 DPI、无衬线字体）
- 优先继承用户已有图表的视觉风格
- 所有图表尽量保持统一的视觉语言
