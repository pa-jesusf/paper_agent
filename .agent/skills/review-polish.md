# Skill: 审校润色

> 全文质量检查、一致性审核和修改建议。

## 触发条件

用户请求检查全文、审校、润色、检查一致性等。

## 检查项

### 自动化检查
- 运行 `tools/glossary_checker.py` — 术语/符号一致性
- 运行 `tools/paper_lint.py` — 综合质量检查

### 人工审阅
1. **术语一致性**: 禁止变体、首现标注英文原文
2. **引用完整性**: `\cite{}` 存在性、图表引用覆盖
3. **引用溯源**: 每条引用都有 `key_quotes` 支撑
4. **写作风格**: 对照 `style-guide.md` 检查
5. **结构同步**: `main.tex` ↔ `outline.md` ↔ 实际文件

## 输出

- `[ERROR]` — 必须修正
- `[WARN]` — 建议修正
- `[INFO]` — 参考建议
